## Current snapshot

- **Tokens package:** consuming `@appdirect/design-tokens@^0.0.6` from Artifactory; `.npmrc` scopes `@appdirect` to the internal registry; VPN required for `npm install`
- **Theme:** `styles/theme.ts` provides Mantine theme + A11Y blue override; `@appdirect/design-tokens` injects `--ad-*` CSS variables via `app/layout.tsx`
- **DESIGN.md:** YAML front matter + prose body. Front matter will be regenerated by `npm run sync-design-docs` once the tokens emitter ships in a published release; body is the source of truth for human semantics
- **Component library:** ~70+ DS wrappers across 11 categories, exported from `@/components/DesignSystem`
- **Branch / consumer model:** GitHub template; downstream prototypes (e.g. `ad-cbp`) cherry-pick updates from this repo
